数据包a

一、项目定位

Mythaicasts 是一个围绕 “生电（红石机械/自动化生产）” 设计的轻量级数据包。

通过进度系统引导玩家逐步建立各类自动化设施，并附带少量实用合成配方与定制附魔，增强生存后期的可玩性与成就感。

二、核心内容总览

  1. 进度系统（任务线）
    以 “获得红石” 作为开端，后续分支覆盖多条生电发展路线：

  2. 自定义配方（部分）
    为部分原版难以量产或获取繁琐的物品提供了合成路径：

方块拆解
紫晶碎片 ← 紫晶块 ×1 → 碎片 ×4

珊瑚块合成
脑纹珊瑚块 / 气泡珊瑚块 / 火珊瑚块 / 角珊瑚块 / 管珊瑚块 ← 对应珊瑚扇 ×4

锁链装备
锁链头盔 / 胸甲 / 护腿 / 靴子 ← 铁粒（代替原版无法合成的锁链甲）

运输矿车系列
箱子矿车 / 熔炉矿车 / 漏斗矿车 / TNT矿车 ← 铁锭 + 对应功能块

马铠系列
皮革 / 铁 / 金 / 钻石 / 铜马铠 ← 对应材料 + 线

其他
附魔金苹果 ← 金块 ×8 + 苹果

荧光墨囊 ← 墨囊 + 荧石粉（无序合成）

  1. 自定义附魔（共 4 种）
    附魔名称 适用装备 效果
    饮血 剑 攻击后为攻击者回复生命（吸血）
    凋落 剑 攻击为目标附加凋零效果（持续 24 秒，等级 01）
    延展 胸甲 增加方块交互距离（每级 +1，最高 3 级）
    不屈意志 头盔 增加 ❤+20 生命值 / 游泳效率 +10 / 护甲 +10 / 护甲韧性 +6

三、内测说明

内测目标
验证进度触发条件是否准确（尤其是潜影盒检测逻辑）

测试附魔在实际战斗中的强度与平衡性

确认合成配方无冲突且符合原版风格

收集整体体验反馈

四、后续计划
可能增加更多支线进度（如猪灵交易所、守卫者农场等）

根据反馈调整附魔数值与配方成本

完善奖励函数，让进度完成更具“获得感”

五、如何参与内测？

内测将于近期启动，届时会在相关社区发布包文件及安装说明。

如有兴趣参与测试或提出建议，欢迎留意后续通知。

生电之路，等你来建。
